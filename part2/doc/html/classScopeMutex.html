Detailed Description

A RAII class for automatic locking and unlocking of pthread mutexes.

This class implements the scope mutex (guard) pattern for pthread mutexes. It locks a mutex when constructed and automatically unlocks it when destroyed, ensuring proper mutex handling even in the presence of exceptions.
